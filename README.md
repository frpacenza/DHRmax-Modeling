# Determiants and predictive modeling of hypoxia-induced maximal heaert rate decline

This repository contains the R scripts used for the statistical analyses performed in my Master's thesis in Mathematics (UniTN &amp; CeRiSM).

## Project description

The project investigates predictors of the reduction in maximal heart rate during acute normobaric hypoxia using physiological data collected from endurance athletes.

The repository includes scripts for:

- data preprocessing
- exploratory data analysis
- statistical modelling
- cross-validation
- figure generation
- model evaluation

## Data availability

The dataset is **not included** in this repository because it is not publicly available.

The data were collected by **CeRiSM (Centro di Ricerca Sport, Montagna e Salute)** and can be requested directly from the institution, subject to their data sharing policies.

## Reproducibility

Since the original dataset is unavailable, the scripts cannot be executed without obtaining the data first.
Some file paths may need to be adapted to the user's local directory structure.

## Software

Analyses were performed using:

- R
- tidyverse
- caret
- glmnet
- randomForest
- lme4
- MuMIn

(and other packages loaded within the scripts)

## Notes

The scripts reflect the workflow developed during the thesis and are shared to improve transparency and reproducibility. While functional, it has not yet been fully organized into a clean analysis pipeline.
